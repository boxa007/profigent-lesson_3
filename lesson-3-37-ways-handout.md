# 37 схем: Claude + Apollo у реальному дні

> 🧭 **Правило великого пальця:** Claude вирішує *хто / чому* → Apollo приносить *список*.
> ⏱️ **Денний ритм:** Ранок — вибір цілей • День — ловля сигналів • Вечір — рекап + навчання.

## Як користуватись цим роздатком (за 30 секунд)

1. Відкрий одну категорію (Morning / Signals / …).
2. Прочитай **жирний** перший рядок кожної схеми.
3. Скопіюй рядок **Workflow** у задачу, автоматизацію або SOP.

**Стратегія засвоєння:** 1 схема на день → за 5 тижнів пройдеш усі 37 → залишиш собі базові 5-7, які реально працюють у твоїй ніші.

> ⚠️ Нагадування з лекції: канал — тільки LinkedIn (invite / like / comment / DM), без email. Схеми з **авто-дотиками** (авто-інвайт, авто-лайк, авто-DM) — це **Tier 2 (advanced, на свій ризик)**. Default — Claude драфтить, ти відправляєш вручну (Tier 1).

# 🌅 Morning — задаємо напрям дня (1-5)

### 1) Вибрати 5 людей для реконекту
**Промпт:** *"З ким говорити сьогодні, якщо хочу можливості, гроші або ріст?"*
**Workflow:** Export LinkedIn connections → import в Apollo → sync з Claude → Claude аналізує зв'язки, відповіді, активність → топ-5 для реконекту.

### 2) Ранжувати сьогоднішній Apollo-pull
**Workflow:** Створити saved Apollo people search → авто-запуск щоранку → результати в Claude → Claude ранжує найвищий сигнал → топ-контакти.

### 3) Вчорашні нотатки → Apollo фільтри
**Workflow:** Зберігати випадкові нотатки в Notes/Notion → авто-надсилати Claude → Claude конвертує ідеї в Apollo фільтри → Apollo будує список людей.

### 4) "З ким би я реально зійшовся?"
**Workflow:** Зберегти LinkedIn профілі → URL в Apollo enrichment → результат у Claude → Claude шукає збіг особистості + інтересів.

### 5) Знайти фаундерів, які будують щось дивне/круте
**Промпт:** *"Find founders building weird/cool things."*
**Workflow:** Claude пропонує незвичні категорії стартапів → конвертує в Apollo company-фільтри → Apollo знаходить фаундерів → Claude ранжує найцікавіших.

# 📡 Signals — ловимо intent (6-10)

### 6) Hiring = buying signal ⭐
**Workflow:** Моніторити hiring-сторінки / LinkedIn jobs (`apollo_organizations_job_postings`) → нові вакансії в Claude → Claude детектить buying signal → Apollo дістає ЛПР.

### 7) Прокрастинація → завтрашній pipeline
**Workflow:** Зберігати випадкові компанії/пости впродовж дня → авто-збір в один список → Claude витягує паттерни → Apollo будує pipeline.

### 8) 37 вкладок → топ-5 пріоритетів
**Workflow:** Browser-розширення трекає відкриті вкладки → URL у Claude → Claude скорить рівень можливості → топ-5.

### 9) "Хто вже шукає допомогу" ⭐
**Workflow:** Моніторити пости фаундерів, найм, новини ("looking for", "need recommendation", "any experience with") → Claude ідентифікує intent → Apollo enrich → warm list.

### 10) Claude знаходить нішу, яка реально працює
**Workflow:** Export розмов/повідомлень → Claude → знаходить повторювані патерни аудиторії → Apollo розширює нішу.

---

# 📥 Capture — моменти у списки (11-15)

### 11) Прогулянка: voice note → Claude → Apollo list
**Workflow:** Записати голосову → Claude транскрибує → конвертує ідеї в Apollo фільтри → Apollo будує пошук.

### 12) Один фаундер → карта схожих людей ⭐
**Workflow:** Зберегти профіль фаундера → `apollo_organizations_enrich` → витягнути traits компанії → знайти схожих фаундерів автоматично (lookalike).

### 13) "Люди в AI, які реально постять" → фільтри
**Workflow:** Ідея в Claude → Claude будує Apollo search-логіку → Apollo знаходить активні профілі.

### 14) Очікування доставки: збудувати нішевий список
**Workflow:** Вибрати нішу → Claude створює фільтри → Apollo будує таргетований список.

### 15) Компанії, які я б сам не шукав
**Промпт:** *"Show me companies I'd never search myself."*
**Workflow:** Claude створює несподівані категорії → Apollo шукає компанії → Claude фільтрує цікаві.

---

# ✨ Vibes — шукаємо енергію, не посади (16-19)

### 16) Опиши "круті стартапи" → отримай реальних людей
**Workflow:** Описати вайб стартапу → Claude визначає traits → Apollo знаходить фаундерів.

### 17) Люди, з якими б випив каву
**Workflow:** Export цікавих профілів → Claude шукає overlap → Apollo розширює схожих.

### 18) Шукай вайби, не індустрії
**Workflow:** Описати вайб → Claude перекладає в traits → Apollo шукає.

### 19) Паттерн "Series A фаундери"
**Workflow:** Export історії розмов → Claude знаходить паттерни → Apollo шукає схожих фаундерів.

---

# ⚡ Fast loops — контент → контакти (20-24)

### 20) YouTube: пауза → компанія → ЛПР
**Workflow:** Згадали компанію → пошук у Apollo → Claude пріоритезує контакти.

### 21) Пост фаундера → схожі люди за 30 сек
**Workflow:** Зберегти пост → Apollo enrich компанію → пошук схожих фаундерів.

### 22) Випадково знайти "інтернет-друзів"
**Workflow:** Export людей, з якими взаємодієш → Claude знаходить спільні інтереси → Apollo розширює.

### 23) Мертвий час у ліфті: мікро-списки
**Workflow:** Вибрати одну нішу → Claude створює фільтри → Apollo будує міні-список.

### 24) Apollo дає контакти, Claude дає reasons ⭐
**Workflow:** Apollo будує список людей → Claude аналізує контекст → додає "1 reason per person why reach out NOW".

# 🔗 Long-term — граф відносин (25-29)

### 25) "Кого майбутній я хотів би знати?"

### 26) Паттерни в тих, хто відповідає

### 27) Нічні сесії цікавості → збережені теми
**Workflow:** Зберегти випадкові ідеї → Claude витягує теми → Apollo шукає.

### 28) Списки за енергією, не посадами
**Workflow:** Описати риси особистості → Claude створює search-логіку → Apollo знаходить.

### 29) "Думають як X, але не відомі"
**Workflow:** Дати Claude один приклад людини → витягнути характеристики → Apollo знаходить схожих.

# 🔍 Clarity — туманне → шукабельне (30-34)

### 30) Туманне → структуроване → шукабельне
**Workflow:** Кинути messy-ідею → Claude структурує → Apollo шукає.

### 31) Doomscrolling → можливості
**Workflow:** Зберігати цікаві пости → Claude витягує сигнали → Apollo розширює список.

### 32) "Не знаю що постити"
**Workflow:** Export даних аудиторії/компаній → Claude знаходить теми → генерує ідеї контенту.

### 33) Знайти людей до того, як стали великими
**Workflow:** Apollo моніторить швидкозростаючі малі компанії (11-50, growth >50% YoY) → Claude детектить breakout-сигнали (funding, hiring rush, press) → outreach ПЕРЕД конкурентами.

### 34) Тижневі дивні тренди в Apollo-даних
**Workflow:** Weekly Apollo export → Claude аналізує → виводить тренди.
