## Категорії як фільтр глибини

- **Точність потрібна (вузький B2B, дорогий чек)** → Signals (6-10) + Behavior (35) + Clarity (30-34). Уникати Vibes.
- **Обсяг потрібен (широка воронка, дешевий чек)** → Vibes (16-19) + Fast loops (20-24).
- **Граф/довгограй (бренд, партнерства)** → Long-term (25-29) + Morning (1-5).

---

## Анти-патерни (коли НЕ рекомендувати)

- Vibes-схеми (#16-19) для вузького дорогого ICP → розмиють точність. Не радити.
- #6 Hiring без реального hiring-сигналу в ICP → порожній результат. Спершу перевір сигнал.
- Більше 3 схем одразу → студент розпорошиться. Завжди ≤3, решта "на потім".
- Tier 2 авто-схеми новачку → ризик банy. Спершу Tier 1 руками.
