# Outreach Architect — скіл Lesson 3

> Твій AI-помічник для вихідних LinkedIn-продажів. Веде за руку: кого шукати → де знайти → як перевірити → що написати.

---

## Що це робить (одним абзацом)

Ти кажеш скілу, кого хочеш у клієнти. Він задає питання, будує портрет ідеального клієнта (ICP), радить, **за якими фільтрами шукати в Apollo**, **як перевірити лідів через Apify** (живі вони чи мертві), і пише тобі **готову цепочку повідомлень у LinkedIn** твоїм голосом. Усе в одному діалозі. Канал — тільки LinkedIn, без email.

**Залізне правило:** Claude думає (кого і чому) → Apollo приносить список → ти робиш дотик.

## Що відбувається всередині (7 фаз, простими словами)

1. **Інтерв'ю** — питає, кого хочеш у клієнти (індустрія, посада, біль, тригер). Якщо вже є контекстна модель — підтягне її, питатиме лише дірки.
2. **Підхід** — каже, з чого почати саме тобі (з найму? з lookalike? зі звуження?).
3. **Схеми** — радить 2-3 з 37 робочих схем + пояснює, чому НЕ інші.
4. **ПЛАН** ⭐ — показує весь маршрут наперед і чекає твоє «так»: *за якими фільтрами шукаємо → чим валідуємо в Apify → яким промптом перевіряємо.* Нічого не запускає без підтвердження.
5. **Пошук** — дістає ~20 лідів з Apollo (або дає фільтри під вставку).
6. **Валідація** — йде в Apify Store, знаходить потрібний actor, показує ціну, пропонує запустити. Відсіює мертвих (зазвичай з 20 → 12-15 живих).
7. **Цепочка** — пише LinkedIn-послідовність по живих лідах: invite → лайк → DM з кейсом → коментар → другий DM. Твоїм голосом. За замовчуванням ти відправляєш вручну (безпечно для акаунта).

## Часті питання

**Скіл нічого не дістає, тільки радить.**
→ Apollo MCP не підключений. Підключи (`/mcp` → Apollo → Authenticate) або працюй у режимі порад (він дасть фільтри, дістанеш сам).

**Цепочка звучить не як я.**
→ Немає `voice-profile.json`. Згенеруй його в Lesson 1, поклади в проект.

**Боюся бана за автоматизацію.**
→ За замовчуванням скіл нічого не відправляє сам — він драфтить, ти тиснеш send вручну. Ризик ~0. Авто-режим (Tier 2) існує, але це окрема історія на свій ризик.

**Apify коштує грошей?**
→ Так, ~$1-3 за валідацію 20 профілів. Скіл завжди показує ціну ДО запуску і чекає твоє «так».
